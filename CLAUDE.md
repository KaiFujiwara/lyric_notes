# Lyrics Notes - 歌詞制作特化メモアプリ

## プロジェクト概要
歌詞のアイデアをその場でメモし、音数判定や韻の確認など歌詞制作に特化した機能を提供するアプリ

## 技術スタック
- **Frontend**: React Native (Expo SDK 54)
- **Database**: ローカルストレージ (Realm/SQLite) → 将来: Supabase
- **Build**: EAS Build (dev/production環境)
- **言語処理**: 形態素解析/音数カウント/韻判定

## ディレクトリ構成
```
/app          # Expo Router画面定義
/components   # 共通コンポーネント
/hooks        # カスタムフック
/lib          # ユーティリティ・ライブラリ
/constants    # 定数定義
/docs         # ドキュメント
```

## 主要機能 (MVP)
1. **プロジェクト管理**: 楽曲単位での歌詞管理
2. **バージョン管理**: 編集履歴の保存
3. **音数カウント**: 行ごとの拍数自動計算
4. **韻チェック**: 末尾母音の一致判定
5. **検索機能**: 単語ベースの検索

## データモデル
- Project: 楽曲プロジェクト
- Version: 歌詞バージョン（セクション別）
- Analysis: 行ごとの解析結果（音数・韻）

## 開発環境
- TypeScript設定済み
- パスエイリアス: @/* → プロジェクトルート
- EAS設定: development/production
