# 画面構成

## 概要
Lyrics Notesアプリは4つの主要画面で構成されており、下部のタブナビゲーションで切り替えが可能です。

## タブ構成

### 1. 歌詞（Projects）画面 (`app/index.tsx`)
**機能**: 楽曲プロジェクトの一覧・管理
- プロジェクト一覧表示
- 各プロジェクトのバージョン数・最終更新日表示
- 新規プロジェクト作成ボタン
- プロジェクト詳細への遷移

**UI要素**:
- ヘッダー（タイトル + 追加ボタン）
- プロジェクトカード一覧（FlatList）
- カード内容：タイトル、メタ情報、矢印アイコン

### 2. フレーズ（Phrases）画面 (`app/phrases.tsx`)
**機能**: 歌詞フレーズのストック・検索
- フレーズ一覧表示
- 検索機能（フレーズ内容・タグ）
- カテゴリ別分類
- タグ表示

**UI要素**:
- ヘッダー（タイトル + 追加ボタン）
- 検索バー（クリアボタン付き）
- フレーズカード一覧
- カード内容：フレーズテキスト、タグ、カテゴリ・日付

### 3. タグ（Tags）画面 (`app/tags.tsx`)
**機能**: タグの管理・統計表示
- タグ一覧表示
- 使用回数統計
- よく使うタグのハイライト表示
- タグ検索機能
- カラーコーディング

**UI要素**:
- ヘッダー（タイトル + 追加ボタン）
- 検索バー
- よく使うタグセクション（横並び）
- 全タグ一覧（使用回数順）
- カラー付きタグカード

### 4. 設定（Settings）画面 (`app/settings.tsx`)
**機能**: アプリ設定・情報・サポート
- 一般設定（通知、自動保存、ダークモード）
- データ管理（バックアップ、同期、エクスポート）
- アプリ情報（バージョン、ヘルプ、評価、応援）
- プライバシー（ポリシー、利用規約）

**UI要素**:
- ヘッダー（タイトルのみ）
- セクション別設定項目
- スイッチ・矢印付き設定項目
- セクションヘッダー

## ナビゲーション構成

### タブナビゲーション (`app/_layout.tsx`)
- **ライブラリ**: Expo Router Tabs
- **アイコン**: Material Icons
- **カラー**: アクティブ #007AFF / 非アクティブ #8E8E93
- **ヘッダー**: 非表示設定

### タブ詳細
| タブ | ファイル | アイコン | タイトル |
|------|----------|----------|----------|
| 歌詞 | `index.tsx` | `library-music` | 歌詞 |
| フレーズ | `phrases.tsx` | `format-quote` | フレーズ |
| タグ | `tags.tsx` | `label` | タグ |
| 設定 | `settings.tsx` | `settings` | 設定 |

## デザインシステム

### カラーパレット
- **プライマリ**: #007AFF（iOS Blue）
- **背景**: #F2F2F7（iOS Light Gray）
- **カード背景**: #FFFFFF
- **テキスト**: #000000（メイン）/ #8E8E93（サブ）
- **境界線**: #E5E5EA

### コンポーネント共通仕様
- **カードデザイン**: 角丸12px、シャドウ付き
- **ヘッダー**: 白背景、下境界線、24ptタイトル
- **検索バー**: グレー背景、角丸10px
- **タグ**: カラー付き背景、角丸12px

### レスポンシブ対応
- SafeAreaView使用（画面上下の安全領域対応）
- FlatList/ScrollView使用（スクロール対応）
- flexレイアウト使用（画面サイズ対応）
