## 🎯 コンセプト

- このアプリは何をしたいのか
    思いついた歌詞をその場でメモする
- 誰のためか（自分／仲間／一般ユーザー）
    自分
- 解決する課題や「こういうの欲しかった」を一言で
    単なるメモではなく、音数判定や韻の確認、単語検索など歌詞に特化したメモ

## 📖 ユーザーストーリー（例）

- 「出先で歌詞のアイデアを思いついたとき、スマホですぐ書き留めたい」
- 「1番と2番の音数をそろえたいときに、自動でカウントして差分を出してほしい」
- 「韻が踏めてるかサクッと確認したい」
- 「この単語の別の言い回しを複数知りたい」

## 🛠 技術要件

- フロント：React Native（Expoベース）
- DB（MVP）：ローカル（Realm/SQLite）
- 同期（将来）：Supabase（Postgres + Auth）
- 音数カウント：形態素解析 or 簡易読み辞書＋正規化
- 韻判定：母音列の一致でMVP、将来は内部韻・頭韻も

## 📂 データモデル（草案）

- Project { id, title, createdAt }
- Version { id, projectId, section, body, createdAt, updatedAt }
- Analysis { id, versionId, lineIndex, moraCount, rhymeTail }

## 🖼 画面イメージ（簡易ワイヤー）

- プロジェクト一覧
- 歌詞編集画面（行ごとに拍数バッジ＋韻マーク）
- フレーズメモ画面

## 🚀 MVPスコープ

- プロジェクト管理
- バージョン保存（簡単な履歴）
- 音数カウント（設定で長音扱い切替）
- 韻チェック（末尾2拍の母音一致）
- 検索（単語ベース）

## 💡 メモ・アイデア

- 歌詞にタグ（失恋、雨、夜…）
- フレーズストック（後で流用できる）
- 音声入力→テキスト化
